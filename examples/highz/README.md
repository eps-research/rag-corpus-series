# High-z Examples

Three worked Jupyter notebook examples for the High-z Kinematic Corpus Z1.

## Available Notebooks

| Notebook | Description | Figure |
|----------|-------------|--------|
| `hz_nb1_rotator_kinematics.ipynb` | J0817 [CII] rotation curve, dispersion profile, V/sigma | `fig_hz_nb1_j0817_rc.png` |
| `hz_nb2_population_diversity.ipynb` | Class fractions, redshift distribution, W15 heatmap | `fig_hz_nb2_population.png` |
| `hz_nb3_eps_omega_bridge.ipynb` | Cross-corpus omega, trilogy table, progenitor candidates | `fig_hz_nb3_eps_omega_bridge.png` |

## Setup

```bash
# Download corpus from Zenodo
# DOI: 10.5281/zenodo.20369286
# Place high_z_kinematic_corpus_Z1.json in this directory

pip install numpy matplotlib jupyter
jupyter lab
```

## Data Required

Download from [Zenodo 10.5281/zenodo.20369286](https://doi.org/10.5281/zenodo.20369286):
- `high_z_kinematic_corpus_Z1.json`
- `high_z_kinematic_corpus_Z1_flat.csv`
